Unsplash api 

Создать приложение для поиска фотографий по апи unsplash https://unsplash.com/documentation

Основная функциональность:
Пользователь должен иметь возможность зарегестрироваться / логинится в приложение( можно использовать локалное хранилище)(использовать oAuth авторизацию)
Пользователь должен иметь возможность искать по фотографии, ставить лайки фотографиям, тем которые были отмечены лайком должны добавлятся в избранное(в локальном хранилище или коллекции). 
Пользователь должен иметь возможность удалить из избранного изображение
Так же в приложении должна быть возможность открытия страницы с самим изображением и комментариями к нему
